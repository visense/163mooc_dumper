# 163mooc_dumper

多线程批量下载网易公开课视频

- 使用线程池机制，并发下载数据
- 支持断点续传（须保持缓存文件的有效性）
- 可达到满速下载

## 参数说明

```
Usage:   163down.py [url]
Example: 163down.py http://open.163.com/special/opencourse/ensteinstheory.html
```
